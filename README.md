#  Kevin- Lin Resume
## Install
Install and read my resume via npm

### in your command line(cl)
$ mkdir kevinlin-resume <br />
$ cd kevinlin-resume <br />
$ npm install kevinlin-resume <br />
$ touch index.js <br />

#### inside index.js:
const resume = require("kevinlin-resume") <br />
console.log(resume) <br />

##### back in your CL
$ node index.js
